##### Dependencies

- Linear algebra backend: BLAS, LAPACK and ARPACK

```sh 
# Install BLAS, LAPACK, and ARPACK
sudo apt-get install libblas-dev liblapack-dev libarpack2-dev
```


- C++ / Python binding : pybind11

```sh
# Install pybind11
pip install pybind11
```
